Controle de Estoque - Java (POO)

Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar os conceitos de Programação Orientada a Objetos (POO) em Java.

O sistema simula o controle de um produto em estoque, permitindo:

Cadastro de produto
Consulta de dados
Entrada de itens no estoque
Saída de itens do estoque

Funcionalidades

 Leitura dos dados de um produto:

Nome
Preço
Quantidade em estoque

Exibição das informações do produto:

Nome
Preço
Quantidade
Valor total em estoque

Atualização do estoque:

Adição de produtos
Remoção de produtos

Loop interativo para múltiplas atualizações


Conceitos aplicados

Este projeto utiliza os principais conceitos de POO:

Encapsulamento
Uso de atributos privados
Métodos getters e setters
Abstração
Classe Produto representa uma entidade do mundo real
Organização em classes
Separação entre lógica principal (Infos) e entidade (Produto)

Como funciona
O usuário informa os dados do produto
O sistema exibe as informações iniciais
O usuário pode escolher:
Adicionar itens ao estoque
Remover itens do estoque
O sistema atualiza e exibe os dados novamente
O processo se repete até o usuário encerrar

Aprendizados

Com este projeto foi possível reforçar:

Manipulação de entrada de dados com Scanner
Uso de métodos e atributos em Java
Boas práticas iniciais de organização de código
Aplicação prática de POO
